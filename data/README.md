## Converting to DOCX and PDF

To convert contracts to Word (.docx) and PDF:

```bash
# Install: pip install -r scripts/requirements-convert.txt
# Install: brew install pandoc wkhtmltopdf  (Mac)

python3 scripts/convert-contracts.py
```

Default: contracts 01,02,03 → PDF; 04,05,06 → DOCX. Output in `data/converted/`. See `scripts/README.md` for details.
